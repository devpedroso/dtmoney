## 🚀 Como executar o projeto
Pré-requisitos: npm/yarn

```bash
# clonar repositório
git clone https://github.com/devpedroso/dtmoney.git

# entrar na pasta do projeto front end
cd dtmoney

# instalar dependências
npm install

# executar o projeto
npm start
```
